# HFGE1181 FA2
Platformer prototype part 2
This project is a 2D Unity game created for my assignment. The game includes player movement, enemies, collectibles, health  UI, score system, pause menu and interactive objects.

The features of the game are:
-player movement
-enemy collision and damage
-health system with hearts
-score collection system
-pause menu
-main menu
-collectible items
-interactive UI
-prefabs and decorations

Tools used are:
-Unity
-C#
-Visual studio

How to play the game:
-Use WASD or the arrow keys to move
-Collect items to increase score
Avoid enemies to prevent loosing health
-Press ESC to pause the game

The project includes:
-Assets folder
-Scripts
-Scenes
-Prefabs
UI elements
